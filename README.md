# Liboschat
Very much in-beta Osmium.chat library. Made with python

Also known as the very first Osmium SDK with voice, and is unofficial.

OpenAI Codex reversed Voice by reading Osmium Web's JS files.

## installation
You cannot install this as a library just yet, you must add it as a git submodule to your client/bot.
You will need aiortc, av, aiohttp.

However you will need to install my [Protodef](https://github.com/usernames122/protodef) because I did NOT want to use protoc.

## NOTES
Please get your own client ID and if possible, use Osmium's MakeABot system instead of using liboschat's selfbot mode.

The spamfilter will be more hostile toward selfbots.

And client IDs should keep your custom client unique.

This library is inspired by discord.py-self.
